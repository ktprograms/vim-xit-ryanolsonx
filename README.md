# [x]it! for Vim

Syntax highlighting for [x]it! for Vim.

## Install

Using [Vundle](https://github.com/VundleVim/Vundle.vim): 

```
Plugin 'ryanolsonx/vim-xit'
```

## Supported highlighting

- Title
- Checkbox
- Simple Tag (like #hello)
